## Quick start

<Steps>
  <Step title="Install Python 3.13 and ddev">
    Follow the [setup guide](/setup) to install the required Python version and the `ddev` CLI tool.
  </Step>
  <Step title="Write your first Check">
    Subclass `AgentCheck` from the base package and implement the `check` method.

    ```python
    from datadog_checks.base import AgentCheck

    class MyCheck(AgentCheck):
        __NAMESPACE__ = 'my_integration'

        def check(self, instance):
            self.gauge('up', 1, tags=['env:prod'])
    ```
  </Step>
  <Step title="Run tests">
    Use `ddev test` to run the integration's unit and integration tests with automatic environment management.

    ```bash
    ddev test my_integration
    ```
  </Step>
  <Step title="Test against a live Agent">
    Start an E2E environment to verify your integration against a real Datadog Agent running in Docker.

    ```bash
    ddev env start my_integration py3.12-1.0
    ddev env test my_integration py3.12-1.0
    ddev env stop my_integration py3.12-1.0
    ```
  </Step>
</Steps>

<Note>
  To add a new integration, open a pull request in [integrations-extras](https://github.com/DataDog/integrations-extras) rather than this repository. This repo is for officially maintained integrations only.
</Note>
